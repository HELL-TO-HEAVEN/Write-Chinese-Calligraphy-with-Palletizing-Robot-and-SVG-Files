# 2017 ARSC Final Project
Path planning for 4 DOFs palletizing robot to write Chinese characters.

## Getting Started

### Prerequisites

Python 3.6
Json
Pygame

### Installing

```
git clone 
cd <directory>
```

## Running the tests

```
python readCharcaters.py
```
